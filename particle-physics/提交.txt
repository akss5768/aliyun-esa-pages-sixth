# 粒子物理模拟器 - 科学实验风

粒子物理模拟器 是一个物理模拟展示项目，采用科学实验风格设计。用户可以观察粒子在重力场中的运动规律，了解基本的物理定律。

## 功能特性

- **粒子模拟** - 模拟粒子在重力场中的运动轨迹
- **参数调节** - 调节重力强度、粒子数量等参数
- **碰撞检测** - 实现粒子与边界的弹性碰撞
- **视觉特效** - 粒子发光、拖尾等视觉效果
- **响应式设计** - 完美适配桌面、平板和手机设备
- **轻量级** - 基于React和Canvas的轻量级实现

## 技术栈

- **前端框架**: React 18
- **动画库**: Canvas API
- **UI样式**: CSS Modules
- **状态管理**: React Hooks
- **数据存储**: LocalStorage
- **构建工具**: Vite
- **环境**: Node.js 18+
