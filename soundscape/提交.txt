# 声景合成器 - 音乐律动风

声景合成器 是一个环境声音合成展示项目，采用音乐律动风格设计。用户可以混合不同环境音效，创造个性化的声景。

## 功能特性

- **环境音效** - 提供多种环境声音选择
- **混音控制** - 调节不同音效的音量平衡
- **预设场景** - 预设多种常用声景组合
- **播放控制** - 开始、暂停、重置等功能
- **响应式设计** - 完美适配桌面、平板和手机设备
- **轻量级** - 基于React和Web Audio API的轻量级实现

## 技术栈

- **前端框架**: React 18
- **音频API**: Web Audio API
- **UI样式**: CSS Modules
- **状态管理**: React Hooks
- **数据存储**: LocalStorage
- **构建工具**: Vite
- **环境**: Node.js 18+
